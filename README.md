# wk1995.github.io

个人主页与博客，聚焦在人工智能与 Android 开发的结合实践。仓库基于 GitHub Pages，`main` 分支内容即为线上展示。

## 站点结构

- `index.html`：首页，展示核心方向、精选项目与博客预览。
- `blog/`：博客列表与文章内容。
  - `index.html`：文章目录页。
  - `ai-on-android-roadmap.html`：端侧推理落地路线图。
  - `designing-ai-experiences.html`：AI 体验设计思考。
- `docs/`：保留 GitHub Pages 默认的 Jekyll 配置，可按需扩展。

## 本地预览

静态页面可直接在浏览器打开 `index.html` 预览；如需使用 Jekyll 主题，可安装 Ruby 环境并执行：

```bash
bundle exec jekyll serve
```
